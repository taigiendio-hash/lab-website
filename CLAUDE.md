# 智慧感測研究室網站 — 專案說明

## 網站目標

這是「智慧感測研究室」的靜態前台展示網站，配有一個後台管理頁面，供不懂程式的老師直接操作新增、修改、刪除網站內容。整體架構為純前端 HTML，資料存放於 Supabase 雲端資料庫（免費方案）。

- **前台** `index.html`：公開展示研究成員、研究主題、最新消息
- **後台** `admin.html`：需使用 Supabase Email 帳號登入，才能對三個資料表進行 CRUD
- **資料庫** Supabase（專案：`zftyiqfqwuhxvibhxosi`）：啟用 RLS，僅允許指定管理者 Email（`taigiendio@gmail.com`）寫入
- **部署平台** Netlify（`courageous-conkies-d21777.netlify.app`），更新方式：把資料夾重新拖曳至 Netlify Deploys

## 語言與文字規範

- 所有介面文字、說明文字、程式碼注解一律使用**繁體中文**
- 數字、日期格式依台灣慣例（民國年不使用，統一西元）

## 視覺風格

- 配色：深藍 `#16365c`（主色）、白底、咖啡色 `#8a5a3c` 點綴線條

- 整體風格：學術、乾淨、現代；不加多餘裝飾

## 合作規則

1. **每次改檔前先說明**：在動任何檔案之前，先用一句話說明打算改什麼、為什麼，等確認後再執行。
2. **不主動新增功能**：只做被要求的事，不因為「可以更好」就自行加功能或重構。
3. **不留多餘注解**：程式碼注解只寫「為什麼」，不寫「做什麼」（好的命名本身就是說明）。
4. **金鑰安全**：`anon`/`publishable` key 可放前端（設計如此）；`service_role`/`secret` key 絕對不放進任何檔案，也不要貼給 Claude。
